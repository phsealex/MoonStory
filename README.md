# Moon Story
## Installing
Drop the `MoonStory` folder inside of `Documents/My Games/Cave Story+/mods` and edit the text file `mods.txt` adding the line:
```
+ MoonStory
```

## Playing the mod

The mod adds a challenge in the `Challenges` menu which acts as an actual separate game mode where you will able to save, change difficulty and even play in Two Player mode!
